# PWA Icons

Le seguenti icone sono necessarie per la PWA:

- `icon-192.png` - 192x192px - Icona standard
- `icon-512.png` - 512x512px - Icona grande
- `maskable-icon-512.png` - 512x512px - Icona maskable (con safe zone)

## Come creare le icone

1. Usa un tool come [PWA Asset Generator](https://github.com/elegantapp/pwa-asset-generator)
2. O crea manualmente con design a tema "microfono verde" (theme color: #10b981)
3. Le icone maskable devono avere contenuto importante entro il 40% centrale

## Placeholder temporanei

Per lo sviluppo, puoi usare icone placeholder generate online o creare SVG semplici.
